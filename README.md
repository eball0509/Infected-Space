# Infected Space

## Project Description

Infected Space is a Mixed Reality (MR) wave-based space shooter developed in Unity. Players are placed into an immersive space combat environment where they must survive against infected enemies attacking from all directions. The game focuses on spatial interaction, shooting mechanics, enemy spawning systems, and immersive MR gameplay using Unity’s XR framework.

This project combines traditional game development with Mixed Reality technologies to create an interactive and replayable combat experience.

## XR / MR Configuration

This project uses Unity’s standard XR setup for Mixed Reality development.

### Required Unity Packages

Make sure the following packages are installed through the Unity Package Manager:

- XR Interaction Toolkit
- OpenXR Plugin
- XR Core Utilities
- Input System

### XR Setup Requirements

1. Open **Edit > Project Settings > XR Plug-in Management**
2. Enable **OpenXR**
3. Under OpenXR Features, enable the appropriate interaction profiles for your target headset
4. Ensure the **XR Interaction Toolkit** package is installed
5. Import starter samples if prompted by Unity
